int rtp_gui_list_size

(void* vplist)

Retrieve the count of entries in a list or menu object.

Documentation

Returns the nuber of entries that have been added to the list or menu by calls to rtp_gui_list_add_item() and rtp_gui_list_add_int_item().

Parameters:
vplist - Handle returned from rtp_gui_list_init()
Returns:
Count of entries in the list or menu.
Preconditions:
rtp_gui_list_init must be called first.

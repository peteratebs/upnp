HTTPCookieInfo* HTTP_FindMatchingCookie

( HTTPCookieContext* ctx,
  const HTTP_CHAR* name,
  const HTTP_CHAR* host,
  const HTTP_CHAR* path,
  int num )

Find a cookie in the database that matches the given host/path

Documentation

There may be a number of cookies that match a given host and path, so this function returns the nth such cookie. This function actually temporarily removes the cookie from the database; so when the caller of this function is done manipulating the cookie, it must be released using either HTTP_ReleaseCookie (which will give it back to the database) or HTTP_DeleteCookie (which deletes it). Until either of these functions are called, the cookie will not be sent along with any outgoing HTTP client requests.

Parameters:
ctx - the cookie database to search
name - option: the name of the cookie; 0 if unspecified
host - the host to match
path - the path to match
num - the index of the cookie
Returns:
pointer to cookie if found, 0 otherwise
See Also:
HTTP_ReleaseCookie, HTTP_DeleteCookie, HTTP_FindExactCookie

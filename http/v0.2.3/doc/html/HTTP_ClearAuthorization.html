void HTTP_ClearAuthorization

( HTTPAuthContext* ctx,
  HTTP_CHAR* host,
  HTTP_UINT16 port,
  HTTP_CHAR* path )

Delete the user name and password for a given server.

Documentation

Clears the user/password credentials from an HTTPAuthContext for the given server/path, if they exist.

Parameters:
ctx - the authentication database
host - the host name
port - the server port number
path - the path on the server
Returns:
0 on success, negative on failure
See Also:
HTTP_AuthContextDestroy, HTTP_SetAuthorization

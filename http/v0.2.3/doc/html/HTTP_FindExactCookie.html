HTTPCookieInfo* HTTP_FindExactCookie

( HTTPCookieContext* ctx,
  const HTTP_CHAR* name,
  const HTTP_CHAR* domain,
  const HTTP_CHAR* path )

Find a specific cookie in the database

Documentation

Locates a cookie that is an exact match for the given name, domain, and path.

This function actually temporarily removes the cookie from the database; so when the caller of this function is done manipulating the cookie, it must be released using either HTTP_ReleaseCookie (which will give it back to the database) or HTTP_DeleteCookie (which deletes it). Until either of these functions are called, the cookie will not be sent along with any outgoing HTTP client requests.

Returns:
pointer to cookie if found, 0 otherwise
See Also:
HTTP_ReleaseCookie, HTTP_DeleteCookie, HTTP_FindMatchingCookie

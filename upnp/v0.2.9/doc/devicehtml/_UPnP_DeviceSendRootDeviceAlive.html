int _UPnP_DeviceSendRootDeviceAlive

( UPnPRootDevice* rootDevice,
  int deep)

Sends alive advertisements for every thing under all root devices.

Documentation

Sends alive advertisements for each device and associated service for all the root devices.

Parameters:
rootDevice - pointer to the root device
deep - if 1, send advertisements for all embedded dives and services
if 0, send advertisements for the device under the root and its associated services
Returns:
error code
